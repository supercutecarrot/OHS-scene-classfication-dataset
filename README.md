# OHS-scene-classfication-dataset
OHS dataset contains 9,050 hyperspectral images with a size of 64Ã—64 and a spatial resolution of 10m. These images are collected from OHS Hyperspectral satellite and labeled into 9 categories, including Farmland, Bareland, forest, industrial, residential, river, road, rural, sealake.
